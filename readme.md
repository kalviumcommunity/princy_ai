# 🧠 Princy.AI  

Princy.AI is an **AI-powered assistant** designed to provide intelligent, real-time support for diverse tasks such as Q&A, content generation, and automation. Built with modern web technologies and integrated with advanced AI APIs, this project demonstrates the power of combining **full-stack development** with **artificial intelligence**.  

---

## 🚀 Features  
- 🤖 **Smart Conversations** – Natural, context-aware chat with AI  
- 🖼 **Text-to-Image Generation** – Generate creative visuals from text prompts  
- 📊 **Data Insights** – Analyze uploaded files or structured data with AI-driven insights  
- 🌐 **Responsive UI** – Sleek, mobile-friendly interface for seamless usage  
- ⚡ **Fast & Scalable** – Optimized backend with real-time updates  

---

## 🛠 Tech Stack  
- **Frontend:** React (Vite), Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB Atlas  
- **AI Integrations:** Gemini API, Hugging Face API  
- **Deployment:** Netlify (frontend), Render (backend)
